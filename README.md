📚 Exercícios em C – Fundamentos

Este repositório reúne exercícios desenvolvidos durante meu aprendizado inicial em linguagem C, com foco na construção de base sólida em lógica de programação e manipulação de dados.

## 📌 Objetivo

Consolidar conceitos fundamentais da linguagem C através de prática progressiva, evoluindo de operações simples até a resolução de um problema mais completo sem auxílio direto.

---

## Conceitos trabalhados

Ao longo dos exercícios, foram aplicados:

- Tipos de dados (`int`, `float`, `char`, `unsigned long`)
- Declaração, inicialização e manipulação de variáveis
- Entrada e saída de dados (`scanf`, `printf`)
- Operadores matemáticos e de atribuição
- Casting de tipos
- Comparações lógicas
- Organização básica de código

---

## 📂 Estrutura

1- hello world.c
2- entradas e saidas.c
3- manipulacao e conversao.c
4- operadores matematicos.c
5- operadores relacionais.c
6- exercicio de media aluno.c
7- exercicio de estoque.c
8- exercicio final - super trunfo.c 

---

Destaque: Super Trunfo
O exercício final consiste em um sistema inspirado no jogo Super Trunfo, onde foi necessário:

Cadastrar dados de duas cartas
Realizar cálculos derivados:
Densidade populacional
PIB per capita
Super poder (combinação de atributos)
Comparar os atributos entre cartas
Exibir os resultados das comparações

Esse exercício integrou todos os conceitos aprendidos anteriormente, exigindo aplicação prática sem roteiro detalhado.

Aprendizados

Durante o desenvolvimento, foi possível reforçar:
Raciocínio lógico aplicado à programação
Atenção à tipagem e conversão de dados
Estruturação de código de forma mais organizada
Resolução de problemas de forma incremental
